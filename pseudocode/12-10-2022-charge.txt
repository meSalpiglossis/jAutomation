Algorithm: Зарядить телефон
1) Найти кабель USB type-C;
2) Обнаружить любой подходящий источник питания;
   DoD: powerbank || адаптер питания || компьютер
3) Подключить к источнику питания;
   if (источник питания = powerbank)
   {
   подключить кабель к powerbank
   DoR: достаточный накопленный заряд
   }
   else if (источник питания = адаптер питания)
   {
   подключить кабель к адаптер питания
   DoR: адаптер питания включен в розетку под напряжением
   }
   else if (источник питания = ноутбук)
   {
   подключить кабель к usb порту ноутбука
   DoR: наличие свободного usb порта с активным питанием 5v
   }
   else
   {
   подключить кабель к телевизору (там всегда свободно :)
   }
4) Подключить кабель другим концом к телефону;
   DoD: заряд телефона показывает > 85%
Result: Заряженный телефон
