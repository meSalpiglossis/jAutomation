Algorithm: Написать алгоритм
1) Создать новый txt файл с названием алгоритма;
   if (идея алгоритма == есть)
   {
   Использовать название, выражающее суть;
   }
   else
   {
   Вспомнить любую регулярную дневную активность, ещё не описанную алгоритмом;
   Использовать название, выражающее суть;
   }
2) Записать название алгоритма;
3) Записать ожидаемый результат выполнения алгоритма;
4) Между названием и результатом записать шаг алгоритма;
   while (вспоминаются шаги осуществления)
   {
   Записывать шаги построчно, расставляя в необходимом порядке, указывая условия;
   }
5) Пронумеровать шаги;
6) Сохранить файл;
Result: Файл с алгоритмом
