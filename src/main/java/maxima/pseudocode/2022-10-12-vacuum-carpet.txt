Algorithm: Пылесосить ковёр
1) Подготовить пылесос; // см. алгоритм "Подготовить пылесос"
2) Включить пылесос на полную мощность;
3) Тщательно пылесосить ковер по направлению от входа в комнату к окну;
   while (участок ковра == ещё не пылесосился)
   {
   Пылесосить участок ковра;
   }
4) Выключть пылесос;
5) Почистить пылесос; // см. алгоритм "Почистить пылесос"
6) Вернуть пылесос на место; // см. алгоритм "Вернуть пылесос"
Result: Пропылесосенный ковер
