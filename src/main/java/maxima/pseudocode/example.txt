Algorithm: Заварить кофе
1) Отфильтровать воду;
2) Набрать в чайник фильтрованной воды;
3) Включить чайник;
Definition of Ready (DoR): В чайнике должна быть ЧИСТАЯ вода (чайник, электричество :))
Definition of Done (DoD): Чайник закипел, остыл до 97 градусов
4) Взять чашку для кофе;
5) Подготовить кофе;
if (кофе == растворимый)
{
ничего не делаем :)
}
else if (кофе == молотый)
{
ничего не делаем :)))
}
else if (кофе == зерновой)
{
перемалываем зерновой кофе в кофемолке;
}
6) Насыпать кофе в чашку;
7) Залить кофе горячей водой;
8) Отфильтровать кофе
if (кофе == растворимый)
{
ничего не делаем :)
}
else if (кофе == молотый || кофе == зерновой)
{
фильтруем через ситечко
}
9) Наливаем кофе в чистую чашку;
Result: Чашка кофе :)
